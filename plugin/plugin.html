Plugin



RandomTP

Ce plugin permet de se téléporté aléatoirement sur son serveur en faisant
/rtp. Si il n'est pas lancé par un joueur il faut faire /rtp et le pseudo du joueur.




CoorPlayer

Ce plugin permet de regarder les coordonées d'un joueur connecté
sur le serveur. Il faut mettre /coplayer et le pseudo du joueur.
